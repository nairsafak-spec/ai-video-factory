# Project Rules

1. **Independent project.** This project is completely independent. It has no connection with NEXORA, EPOXAR, or any previous project.
2. **Open source first.** Open-source software is always the preferred choice.
3. **Self-hosted.** The system must run on our own server.
4. **Quality over speed.** Code quality is more important than delivery speed.
5. **Modular by design.** Everything should be built as modular, independent components.
6. **Document decisions.** Every important decision must be documented.
7. **Long-term goal.** Build a fully autonomous AI Video Factory.
